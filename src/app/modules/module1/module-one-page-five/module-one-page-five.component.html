Extra HTML Functions

Finally, we have the additional HTML functions and tags that we’ll cover here. One really important additional function is adding images to your webpage. This can be done with an image tag, denoted by: <img>.

This tag only uses the opening tag, and has two attributes that you definitely want to fill out: the src attribute and the alt attribute. The src attribute defines that image source, or in plain terms, the image file that is going to be used in that tag. The alt attribute defines some text that should be displayed if the image cannot be loaded for one reason or another. For example, if you wanted to upload an image called “castle.jpg” you might want to create an image tag that looks like this:

<img src=”castle.jpg” alt=”A really cool castle I found online”>

There is also another tag that you should learn about: the <a> tag (also known as the anchor tag). This tag is typically used to create links that take you to other web pages or websites.

Have you ever seen that blue text that’s underlined and it takes you somewhere else on the internet (hint you have definitely seen this on the second page of this module)? If you have, that was likely done with an <a> tag.

To create a working <a> tag, you would want to have some sort of text in between the opening and closing tags, and then you also need to specify the destination that you want to take the user to when they click on the link. To do this, you need to set the ‘href’ attribute like this:

<a href = “https://www.google.com”> Do you want to go to Google? </a>.

Additionally, if you want the link to open a new tab instead of redirecting in the current tab, add an additional attribute after href called ‘target’ and set it to ‘_blank’. For example:

<a href = “https://www.google.com” target=”_blank”> Do you want to go to Google? </a>

The <a> tag is another powerful tool in your arsenal of HTML tags to make a high quality website.
