All About Tags!

Tags are the basic building blocks of HTML and let us specify a lot of things about what that building block is supposed to do or contain.

For example, we could use a paragraph tag to put regular text into. The way that tags are formatted is by having the tags ID wrapped in “<” “>” characters. Going back to the paragraph tag example, you would type “<p>” to place an opening paragraph tag.

Another important thing about tags is that you have an opening and closing tag, similar to how you have opening and closing parentheses like this: “(” “)”. An opening tag simply has the ID within the brackets like this: <p>, however the closing tag has a forward slash in the first angle bracket like this </p>. A full set of opening and closing tags for a paragraph tag should look like this: <p> </p>.

These tags just help us define what should be held within the tags, just like how with parentheses the actual content is held within them. Similarly, with tags you just put the content that you would want in them between the tags. For example, you could write <p> I love HTML! </p> and the text “I love HTML !” would be displayed on the web page.

Building off of this, you can even have the opening and closing tags be on separate lines and nothing would necessarily change. The results would still be the same as if you wrote everything in one line, your code is just more readable, which is an important thing to consider.
