On the Topic of Headers

The header tags are used to create headings and sub-headings for the pages you create. There are many different kinds of header tags, that range from header 1 to header 6, where each time you go up a header number the text will get smaller and less prominent. This is because we start with header 1, which should be the main title of your web page. There should only ever be one header 1 present on a given page, no more.
