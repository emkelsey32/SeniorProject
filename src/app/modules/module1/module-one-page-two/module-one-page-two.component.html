What's all this Spacing?

Spacing is super important for any web page, it’s one of your best tools to make sure the content on the page is readable for everyone. There are few different parts that make up spacing, and it’s important to know where each one lies.

As you can see from the diagram, the margin is the outermost layer, the border is the middle layer, and the content is the innermost layer. The top, left, right, and bottom of each element can be changed individually to increase or decrease the spacing between other elements.

For example, let’s say that you have two lines of text and a picture on your web page, and the picture is supposed to be to the right of the text. Let’s also imagine that the picture needs to be farther away from the text, but the second line needs to be closer to the first line. To make these changes, you could set the bottom margin of the first line, or the top margin of the second line, to be smaller. For the picture, you could set the right margins of the text to be larger than what it currently is, and these changes would accomplish the original goals.

Another helpful tool for spacing is the line break tag: <br />. This tag breaks the current line so you can start fresh on a new one. Here is an example of this:

<p> Sample text here </p> <br /> <p> Sample text but on a new line </p>.
