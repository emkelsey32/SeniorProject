Libraries

Libraries are another important tool that can help to save you countless hours of headache because now you get to use code that someone else made that they know works. There are libraries for almost everything that you could think of, and this can be super helpful for a variety of reasons, however there is one main reason that stands out above all the rest: not having to reinvent the wheel.

Take a second to imagine if every car company had to take the time to recreate how a wheel works every single time they wanted to make a new car. They would probably be significantly worse at creating a wide variety of new cars every year, which is exactly why they wouldn’t do that. Instead, they get to keep that knowledge of how to make the wheel, and even the actual wheels, to be able to use in other projects. Libraries let you do the same thing.

You may be in a situation where you are working on something very common that someone else surely has created before, but it takes you 10 hours to create it. You then want to start another project, and you have to spend another 10 hours recreating the same thing. That would not be very efficient, nor would you have a good time doing it if you’re like most people. Using a library would let you skip both of these steps by just letting you use a version that someone has created and included in the library.

And they usually don’t just include one thing, the reason they’re called libraries is because not only do they include all the files necessary to make one thing work, but they include everything to make a whole bunch of things work. For example, look at the bootstrap library and see how much is included in just this one thing.

In the creation of this course, multiple libraries were used to be able to quickly and efficiently create and use components that make our application feel much more professional. These libraries include things such as bootstrap, but also materialUI, and Liveweave. We encourage you to do your own research on libraries to see how useful they can be.

We aren’t going to go too much more in depth about libraries, since that’s what the about page of a library is for, but we wanted to talk about this because we found it to be an important topic that everyone learns about given how powerful they can be. In your future projects, consider investigating a library to help yourself save time and energy by taking advantage of the resources out there on the internet.
