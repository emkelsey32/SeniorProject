First, we will start with the float property. This property allows you to move an element to the left or right of the containing element. This can be used to create layouts with multiple columns. This property has three values it can be set to: left, right, or none. With a float value of none, the object would appear on a different line than everything else. Next we’re going to talk about positioning. What is positioning you might be wondering? It is a way to change the location of an element, which is different from spacing, and there are four main types that we’re going to talk about here: static, relative, absolute, and fixed. Static positioning is the default method, and just means that the element is positioned according to the natural flow of the document. Relative positioning allows you to move an element relative to its current location. Absolute positioning allows you to move an element to a specific location on the page. And finally, fixed positioning allows you to move an element to a specific location on the page and keep it in place even when the user scrolls the page.

The final topic that we’re going to cover for CSS basics is grid systems. Grid systems are a great way to create responsive layouts, and one such grid system known as Flexbox, is a new grid system that is even more versatile than previous ones. Flexbox was specifically designed for responsive layouts, and as such is more flexible than the standard CSS grid. Understanding both however is key to creating layouts that look good on all devices, as something you will have to consider when making a website is how it looks on the three main technologies that access the internet: computers, tablets, and mobile devices. A good resource to learn about these grid systems can be found at https://web.dev/learn/css/grid/.
