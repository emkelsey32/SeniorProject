Freaky Fonts

Fonts are another important aspect of CSS and it has a variety of properties that you can modify. Choosing the right font can have a huge impact on how readers experience and view a website. If you have a super silly looking font, it would likely make it harder for a user to take your website seriously. This could be a huge issue depending on the purpose and tone you would like to set with your application.

Additionally, you want to ensure that you select a font style that is easy to read, as a difficult to read website is objectively less effective. To give you a quick glance at some of the properties that you can modify with the font, look at the image below:

The font style is broken down into 3 categories: normal, italic, and oblique (which is similar to italics but not quite the same). Font weight has two categories: normal and bold. Font size has two types of sizing that you can use: absolute size and relative size. Absolute size sets the values to be unchanging for any reason. Typically, this is not usually desirable, and is measured in pixels. Relative size is a much better tool, and allows a user to set the size relative to one of a few different measurements like the size of your window or screen. This is much better in terms of accessibility and allows for your web page to fit on screens of multiple sizes, including something like your phone!

Another neat thing you can do with CSS to modify text is set the alignment to something like the left, center, or right, and change the line spacing to make body content more readable.
