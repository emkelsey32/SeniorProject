Clone and Add

To get started with GitHub, you can navigate to github.com and create an account here. Then on the page with all of the projects, click on the green button labeled New.

Follow the steps to create a new repository such as selecting a name and a description for the project. After creating the new repository, copy the link in the blue box. Open up a terminal, using “cmd” and use cd commands to get to the directory where you want the project to be. Read about cd here: https://learn.microsoft.com/en-us/windows-server/administration/windows-commands/cd.

We recommend making a new folder for the repository with a name that will match the project. Once inside this folder in the terminal, run git clone [INSERT YOUR URL] .

This will connect the repository to this folder on your computer.

Next, you need to create files, (maybe an HTML and CSS file) and add code to them. When you are ready to send your changes to the repository. Open up the terminal, cd in to the directory, and run “git add .”

This command will add all of your files to the staging section of git, labeling them as files that are ready to be added to a commit, which we will talk about in the next lesson.
