SPA Day

Another important topic to cover is single page applications (SPAs), which many frameworks like Angular and React can create. The way that a SPA works is by having all of your application’s functions exist within a single page and then conditionally rendering the things that matter to the user rather than loading an entirely new page.

This has the potential to massively improve the user’s experience. Rewinding a little bit though, the ‘conditionally rendering’ part of a SPA means that the things loaded on the screen only happen when they’re needed. The reason everything is allowed to work despite all being on a singular page is that the application will use different routes throughout the program to find and render the things that are needed based upon what you tell it to render at that route.
